Utilities
=========

The following Utilities are used for technical purposes and are not required for the basic usage of promptzl.
However, to understand the most important utilities of this package, the most essential ones are listed below.
A noteworthy function is :meth:`~promptzl.utils.calibrate`, which is used to calibrate the
probabilities, sometimes leading to better performance. More details about calibration are described in :ref:`calibration`.
